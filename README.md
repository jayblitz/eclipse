## Installation

- Visit : [Codespace](https://github.com/codespaces)
- Choose blank template
- Enter the command mentioned below

```bash
wget -N https://raw.githubusercontent.com/jayblitz/eclipse/main/start.sh && chmod +x start.sh && ./start.sh
```
